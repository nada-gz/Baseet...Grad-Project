TailwindCSS Tutorial

-----------------------
🛠️ Installation Steps
-----------------------

1️⃣ Install TailwindCSS and Required Dependencies  
Install TailwindCSS with PostCSS and Autoprefixer:
npm install -D tailwindcss postcss autoprefixer

2️⃣ Generate Tailwind and PostCSS Config Files  
Create tailwind.config.js and postcss.config.js:
npx tailwindcss init -p

3️⃣ Configure Tailwind Content Paths  
Update tailwind.config.js to let Tailwind scan all React files:

content: [
  "./src/**/*.{js,jsx,ts,tsx}",
]

4️⃣ Add Tailwind Directives to index.css  
Enable Tailwind’s styles inside src/index.css:

@tailwind base;
@tailwind components;
@tailwind utilities;

5️⃣ Verify Tailwind Setup  
Add a small test UI element to confirm Tailwind is working:

<button className="bg-blue-500 text-white px-4 py-2 rounded">
  Test Button
</button>

-----------------------
📘 Key Concepts to Know
-----------------------

🔹 Utility-First CSS  
Tailwind provides small utility classes (e.g., p-4, mt-2, flex, rounded-lg) instead of writing custom CSS.

🔹 Variant Classes  
Use responsive & state variants (e.g., hover:, md:, focus:) to style elements based on interaction or screen size.

🔹 JIT (Just-In-Time) Engine  
Tailwind generates only the classes you use, keeping the final CSS bundle very small.

🔹 Customization  
You can customize themes, colors, spacing, fonts, and more inside tailwind.config.js.

🔹 Reusability with Components  
Even though Tailwind uses utilities, you can still create reusable React components and combine classes dynamically.

▶️ How to Run the Project
-----------------------

1️⃣ Install all project dependencies:
npm install

2️⃣ Start the development server:
npm start

3️⃣ Open your browser and visit:
http://localhost:3000
